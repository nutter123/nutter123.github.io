# 全局安装

```bash
npm install -g hexo-cli
```

# 初始化

```bash
hexo init blog
cd blog
npm install
```

# 本地预览

```bash
hexo server
```

# 新建文章

```bash
hexo new "新文章"
```

# 生成静态文件

```bash
hexo generate
```

# 部署

```bash
hexo deploy
```
